## Installation

first you need python3 to run this prgram

1. install python
```bash
  $sudo apt install python
```
2. clone git repo
```bash
  $ git clone https://github.com/Pureparadise56b/managefiles.git
```
3. change directory
```bash
  $ cd managefiles
```
4. run main.py file
```bash
  $ python3 main.py
```
